To deploy an existing application:

1. Configure a MySQL instance
2. Run database/scripts/db_create.sql on the MySQL instance.
3. Read config/README.txt regarding the configuration files.
4. Copy aggredata.log4j.xml to your application container's home directory
5. If you need to change the default JDBC settings, copy aggredata.properties
    to your application container's home directory.
6. Deploy aggredata.war to you application container
7. Go to http://<container ip address>:<port>/aggredata to access the
   application (e.g. http://127.0.0.1:8080/aggredata
8. The default username/password is admin/admin
